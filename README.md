# Iris Flower Classification

## Project Overview
This project uses Machine Learning to classify Iris flowers into three species:

- Iris-setosa
- Iris-versicolor
- Iris-virginica

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn

## Machine Learning Model
